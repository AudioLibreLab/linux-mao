🎸 Synthèse : Technique de Démanché
1. Les Règles de Direction
L'objectif est d'anticiper la position pour ne jamais être "à court de doigts".

Vers le bas (Aigus / Corps) : Utiliser l'auriculaire (4) ou l'annulaire (3).

Pourquoi ? Cela déplace la paume vers les notes hautes et libère les autres doigts derrière.

Vers le haut (Graves / Sillet) : Utiliser l'index (1) ou le majeur (2).

Pourquoi ? Cela repousse la main vers la tête du manche, préparant l'extension vers les graves.

2. Routine d'Entraînement
Appliquer ces démanchés sur trois structures types pour couvrir tout le manche :

Gamme Brisée : Travail par intervalles (ex: en tierces Do-Mi, Ré-Fa).

Gamme Brodée : Autour d'une note pivot (ex: Do-Ré-Do).

Arpèges : Sauts de notes (1-3-5-7) pour gérer les grands déplacements.

3. Points de Vigilance
Le Pouce : Il doit glisser de façon fluide derrière le manche, jamais rester ancré.

Le Regard : Toujours viser la case d'arrivée avant le mouvement.

Le Son : Relâcher la pression sans quitter la corde pour éviter les bruits de "glissando" non voulus.
